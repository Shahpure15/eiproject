PROJECT OVERVIEW

The SmartStreet system integrates LDR, PIR, and MQ-135 sensors with Arduino to create an intelligent street lighting solution with real-time air quality monitoring. Data is sent to ThingSpeak cloud for persistent storage and visualization.

KEY FEATURES

Smart Lighting Automatic control based on light + motion
Energy Efficient Saves up to 68% energy using motion detection
Pollution Monitoring Real-time harmful gas detection & alerts
Cloud Connected Interactive dashboard with live & historical data
